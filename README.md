# videoGameS3
Video game development project for the C/C++ advanced programming course

## Installation avec SDL2
```sh
git clone https://github.com/victord54/videoGameS3.git &&
cd videoGameS3 &&
mkdir build &&
make
```
## Installation avec SFML
```sh
git clone https://github.com/victord54/videoGameS3.git &&
cd videoGameS3 &&
mkdir build &&
git checkout SFML &&
make
```
