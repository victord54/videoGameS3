# videoGameS3
Video game development project for the C/C++ advanced programming course

## Clone with SDL2 version
```sh
git clone https://github.com/victord54/videoGameS3.git &&
cd videoGameS3 &&
mkdir build &&
make
```
## Clone with SFML version (the best version)
```sh
git clone https://github.com/victord54/videoGameS3.git &&
cd videoGameS3 &&
mkdir build &&
git checkout SFML &&
make
```
